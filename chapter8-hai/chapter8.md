# 第八章 网

SMA 毕业以后，我在网上写一些文章。

写技术，也写一些零碎的想法。那时候互联网刚刚热起来，很多人把东西放到网上，也不知道会被谁看到。

有一天晚上，我在宿舍上网。

那时候还没有微信，很多人用 MSN。电脑屏幕亮着，我一边看网页，一边刷邮件。忽然右下角弹出一个窗口，有人加我为好友。

对方的名字叫：

“天空”。

头像是一个大肚皮的财主。

我以为是谁随便加人。

过了一会儿，对方发来一句话：

“我看过你写的文章。”

接着又发了一句：

“我是丁磊。”

我没太当真。

他说，他最近在看 Python 的东西，想让我整理一下 Python 的开源项目。

那时候 Python 还没有现在这么流行。国内很多程序员还在用 C++ 或 Java，Python 很少有人谈。

我想了一下，还是开始搜。

那时候 Google 还很好用。我把所有和 Python 有关的页面几乎都翻了一遍。开源项目、邮件列表、论坛帖子，一条一条整理出来。

等整理完的时候，已经很晚了。

我把列表发给他。

过了一会儿，对方回了一句：

“不错。”

后来我才知道，那个人真的是丁磊。

那一年大概是二〇〇三年前后。
丁磊已经是中国首富。

后来他让我去网易。

公司在广州。我第一次去的时候，对互联网公司其实没有什么概念。到了公司以后，有人带我在办公室里转了一圈。

问题是，我刚进去的时候没有工位。

也没有电脑。

公司那几年发展很快，人一批一批进来，很多事情还没完全安排好。

后来丁磊说：

那你先坐我办公室吧。

我以为只是随口一说。

结果他们真的把我带过去了。

房间不大，一张桌子，一台电脑，几把椅子。墙上挂着一幅字。

五个字：

「云层之上无风雨。」

我就坐在那里上网、写代码、干活。

有时候抬头，就能看见那幅字。

丁磊很少真正坐在办公室里。他经常在公司各个地方转。有时候在工程师那边，有时候在会议室。

很多时候，已经很晚了，他还在公司。

有一次我晚上还在楼里，看见他从会议室出来。走廊很安静，只剩下一些电脑屏幕的光。

那时候他三十出头。

有一天晚上，我们一起出去吃饭。

不是去什么好地方，只是街边的小馆子。门口一口锅，老板在下面条，热气一阵一阵往外冒。

我们坐下来吃面。

说的话不多。

吃完以后，我抢着把单买了。

他是首富，我是新来的。

但我觉得这顿饭应该我请。

后来他让我做游戏。

那几年网易的游戏刚开始起来，很多人都在做。

但我当时觉得，自己是研究生毕业，做游戏好像不太“学术”。

现在回头看，这个想法其实很幼稚。

那时候我还想继续读书。

有一次我跟他说，我想出国读博士。

他说：

书读多了没什么用。

我当时听进去了。

于是没有去。

这件事后来一直留在心里。

还有一件事，我也一直记得。

那时候我还没有女朋友。

有一次我跟他说，能不能帮我介绍一个。

他说，好啊。

过了几天，公司有人来找我，说安排我见一个人。

结果见到以后才发现，是一个女上司。

后来我才知道，人家已经结婚了。

我当时很生气。

觉得他说话不算数。

现在回头看，那时候我其实很不成熟。

我在网易待的时间不算很长。

那几年我一直在想一个问题：人为什么要工作。

不是为了挣钱，而是更深一点的东西。

我总觉得，世界应该更大一点。

所以后来我离开了网易。

离开的时候，没有什么特别的场面。只是收拾东西，从办公室走出来。

走廊还是一样安静。

很多人还在写代码。

很多事情还在继续。

但有一件东西留在了我心里。

那就是那幅字。

「云层之上无风雨。」

很多年以后，我还会想起那间办公室。

想起我坐在那里上网、写代码，偶尔抬头，看见墙上的那五个字。

互联网把我带进了那间办公室。

后来又把我带去了很多地方。

但那幅字，一直没有离开过。
