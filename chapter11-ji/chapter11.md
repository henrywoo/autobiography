# 第十一章 云

从谷里出来，我去了 Oracle，做云上的 AI。OCI，Oracle Cloud Infrastructure，在云里建基础设施，在基础设施上跑 AI。我见过 Larry Ellison，他娶了个中国太太，我们在二〇二五年前后把 Oracle 的股价从五十块做到三百五，那段时间他一度超过马斯克成为世界首富。但更让我想的是另一件事：如果机器可以写代码，人为什么还要工作？在云上做 AI 的时候，这个问题越来越具体。云在上面，人在下面，代码在中间。你写代码，机器也在「写」代码。云层之上无风雨，可云层之下，我们还在奔波。这一章叫云，是因为路走到了这里：你在云里做事，也在想，人到底该站在哪里。
