# 第八章 网

> 云层之上无风雨。

SMA的課程快要結束了，大家面臨工作還是讀博士的問題。

在學校待了快20年，對工作一無所知。

我們這屆是國家不包分配的第一屆，之前的師兄師姐都沒有要自己找工作的說法。

工作到底是個什麼東西？我甚至不知道爲什麼人要工作？

我聯繫了一個教授準備去讀博士，同時同學給介紹了一個研究所研究工程師的職位，所以就在國大NUS待着沒有任何壓力。

業餘時間非常無聊，只好寫點文章做做Flash打發時間。[^2]

其實在SMA還沒有毕业的時候，我就在网上比如CSDN写一些文章。有時候還幸運的被推到CSDN首頁。我也有一些紅心讀者網絡粉絲。

写技术，也写一些零碎的想法。那时候互联网刚刚热起来，很多人把东西放到网上，也不知道会被谁看到。

有一天晚上，我在宿舍上网。

那时候还没有微信，很多人用MSN。[^3] 电脑屏幕亮着，我一边看网页，一边刷邮件。忽然右下角弹出一个窗口，有人加我为好友。

对方的名字叫：

“天空”。

头像是一个大肚皮的财主，笑得有點誇張。

我以为是谁随便加人。

过了一会儿，对方发来一句话：

“我看过你写的文章。”

接着又发了一句：

“我是丁磊。”

他说，他最近在看Python的东西，想让我整理一下Python的开源项目。

那时候Python还没有现在这么流行。国内很多程序员还在用C++或Java，Python很少有人谈。

我不知道丁磊是誰，也没太当真，因爲當時我的紅心讀者很多。但我想了一下，还是开始搜。

那时候Google收錄的網頁很少，还不是很好用。

我搜到這個人是網易的老闆，看來不像是騙人的。

第二天，我把所有和Python有关的页面几乎都手動翻了一遍。

Google很小，整個給Python相關的內容也不算太多。

我把开源项目、邮件列表、论坛帖子，一条一条看了再整理。

整理完之後，我把列表发给他。

他回了一句：

“不错。”

我不確信他到底有沒有一條條看我的列表，但后来我知道，那个人是當年的中國首富。[^1]

他在給遊戲引擎找開發人員。

那一年大概是二〇〇三年前后,我23歲，他32歲。

后来他让我去网易。

公司在广州。我第一次去的时候，对互联网公司其实没有什么概念。到了公司以后，有人带我在办公室里转了一圈。

问题是，我刚进去的时候没有工位。

也没有电脑。

公司那几年发展很快，人一批一批进来，很多事情还没完全安排好。

后来丁磊说：

那你先坐我办公室吧。

我以为只是随口一说。

结果他们真的把我带过去了。

房间不大，一张桌子，一台电脑，几把椅子。墙上挂着一幅字。

五个字：

「云层之上无风雨。」

我就坐在那里上网、写代码、干活。

有时候抬头，就能看见那幅字。

丁磊很少真正坐在办公室里。他经常在公司各个地方转。有时候在工程师那边，有时候在会议室。

很多时候，已经很晚了，他还在公司。

有一次我晚上还在楼里，看见他从会议室出来。走廊很安静，只剩下一些电脑屏幕的光。

那时候他三十出头。

有一天晚上，我们一起出去吃饭。

不是去什么好地方，只是街边的小馆子。

广州的夜里还有点热，我们坐在门口的塑料凳上。

门口一口大锅，老板在下面条，热气一阵一阵往外冒，混着酱油和葱花的味道。

丁磊穿着很普通，和白天在公司里差不多。

我们各要了一碗面。他问我习惯不习惯广州，我说还行。

他又说，公司现在人多了，很多事情他管不过来，你好好做就行。

话不多，面吃得很慢。

旁边有几桌人，大声说着粤语，我们这边反而安静。

吃完以后，我抢着把单买了。

他是首富，我是新来的。

但我觉得这顿饭应该我请。

他也没争，说了声谢，我们就往回走。

街上的灯不太亮，两个人并排走，谁也没再提公司的事。

那是我第一次和他在公司外面吃饭，也是印象最深的一次。

后来他让我做游戏。

那几年网易的游戏刚开始起来，很多人都在做。

但我当时觉得，自己是研究生毕业，做游戏好像不太“学术”。

现在回头看，这个想法其实很幼稚。

那时候我还想继续读书。

有一次我跟他说，我想出国读博士。

他说：

书读多了没什么用。

我当时听进去了。

于是没有去。

这件事后来一直留在心里。

还有一件事，我也一直记得。

那时候我还没有女朋友。

有一次我跟他说，能不能帮我介绍一个。

他说，好啊。

过了几天，公司有人来找我，说安排我见一个人。

结果见到以后才发现，是一个女上司。

后来我才知道，人家已经结婚了。

我当时很生气。

觉得他说话不算数。

现在回头看，那时候我其实很不成熟。

我在网易待的时间不算很长。

那几年我一直在想一个问题：人为什么要工作。

不是为了挣钱，而是更深一点的东西。

我总觉得，世界应该更大一点。

所以后来我离开了网易。

离开的时候，没有什么特别的场面。只是收拾东西，从办公室走出来。

走廊还是一样安静。

很多人还在写代码。

很多事情还在继续。

但有一件东西留在了我心里。

那就是那幅字。

「云层之上无风雨。」

很多年以后，我还会想起那间办公室。

想起我坐在那里上网、写代码，偶尔抬头，看见墙上的那五个字。

互联网把我带进了那间办公室。

后来又把我带去了很多地方。

但那幅字，一直没有离开过。

[^1]: 丁磊，网易创始人，2003 年前后曾为中国首富。

[^2]: Flash，Adobe 推出的多媒体创作与播放平台，2000 年代曾广泛用于网页动画、游戏与互动内容，2020 年底已停止更新与支持。

[^3]: MSN Messenger（后更名为 Windows Live Messenger），微软推出的即时通讯软件，2000 年代在中国及全球广泛使用，2013 年除中国大陆外多数地区已关闭，2014 年中国大陆服务亦停止。
