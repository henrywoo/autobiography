## 天空

SMA 的课程快要结束的时候，大家开始讨论一个问题。

工作，还是读博士。

很多同学已经联系好了教授，有人准备去美国，有人打算留在新加坡做研究。

教室里常常能听见这些话题：申请、推荐信、实验室、funding。

我坐在旁边听着，却没有什么真实的感觉。

在学校待了快二十年，我对工作几乎一无所知。

甚至不知道，人为什么要工作。

之前的师兄师姐，大多是国家分配。

毕业以后去研究所、大学、部委，一切都已经安排好。

我们这一届不一样。

国家不再包分配，每个人都要自己找路。

那时候我一边联系教授，准备读博士；

一边又听同学介绍，说新加坡有研究所招工程师。

两条路都摆在面前。

但我其实并不着急。

那段时间，我在国大待得很闲。

课不多，实验也不多。

白天在校园里晃一晃，晚上回宿舍上网。

新加坡的夜很安静，宿舍楼外面常常只有风声和远处公路的车声。

实在无聊的时候，我就写点东西。

写技术，也写一些零碎的想法。

那时候互联网刚刚热起来。

很多人开始把文章放到网上，也不知道会被谁看到。

我在 CSDN 上写过一些技术帖子，有时候还会被推到首页。

慢慢也有一些读者，给我留言、发邮件。

那种感觉有点奇怪。

你坐在一个很小的房间里，对着一块发光的屏幕敲键盘。

就像投个漂流瓶到海里，不知道这些文字会跑到哪里去。

\
有一天晚上，我像往常一样在宿舍上网。

电脑屏幕亮着，桌子上只有一盏台灯。

外面的走廊已经很安静，同寝室的人都睡了。

那时候还没有微信。

很多人用的是 MSN。

我一边看网页，一边刷邮件。

忽然，屏幕右下角弹出一个小窗口。

有人加我为好友。

名字只有两个字：

天空。

头像是一个大肚皮的财主，笑得有点夸张。

我以为是谁随便加人。

过了一会儿，对方发来一句话：

“我看过你写的文章。”

我没有马上回。

那时候在网上认识陌生人很常见，很多人只是随便聊几句。

过了大概半分钟，对方又发来一句：

“我是丁磊。”

他说，他最近在看 Python 的东西.

想让我整理一下Python的开源项目。

那时候 Python 还没有现在这么流行。

国内很多程序员还在用 C++ 或 Java，Python 很少有人谈。

我一开始并不确定他说的是真是假。

于是我打开浏览器去搜索这个名字。

那时候 Google 收录的网页并不多，搜索也没有现在这么方便。

我翻了几页，才慢慢看明白。

这个人，是网易的老板。

看起来不像是在开玩笑。

第二天，我把所有和Python有关的页面几乎都手动翻了一遍。

那时候关于 Python 的资料其实并不多。

开源项目、邮件列表、论坛帖子，我一条一条看过去，再慢慢整理。

整理完以后，我把列表发给了他。

他很快回了一句：

“不错。”

我不确定他有没有真的把那些东西看完。

但后来我才知道，那个人是当年的中国首富。

那一年大概是二〇〇三年前后。

我二十三岁，他三十二岁。

后来他问我愿不愿意去网易。

其实那时候工作并不难找。

真正让我困扰的，是另一件事。

我还没有女朋友。

家里的观念很简单：

读书的时候绝对不能谈恋爱，毕业以后却希望马上抱孙子。

我对丁磊说：

如果能给我介绍个女朋友，我就去。

没想到他居然答应了。

还说可能是南航的空姐。

那几天我甚至有点睡不好觉。

那时候我还没有真正经历过职场.

他也还是一个刚从创业早期走出来的人。

很多事情都还带着一点随意。

于是我们就这样说定了。


## 广州

网易的总部那时候在广州天河工业园。

到了公司以后，丁磊带我在办公室里转了一圈。

他还把我介绍给了当时的COO麦克。

那天是周末，我带着自己的笔记本电脑。

公司那几年发展很快，人一批一批进来，很多事情都还没完全安排好。

我一时没有工位。

丁磊想了想，说：

那你先坐我办公室吧。

我以为只是随口一说。

结果他们真的把我带去了总裁办公室。

房间不大。

一张桌子，一台电脑，几把椅子。

墙上挂着一幅字。

五个字：

“云层之上无风雨。”

我就坐在那里上网、写代码。

有时候抬头，就能看见那幅字。

\
丁磊其实很低调，低调到有些员工甚至不认识他。

有一次我在办公室写代码，进来一个打扫卫生的阿姨。

我站起来让她打扫桌子下面。

她说：

你是老板，不用站起来。

我说我不是。

这里的老板叫丁磊，中国首富。

她听了以后摇摇头，说她不认识。

我当时有点愣住。

后来想想，这件事其实也很正常。

总之，我成了“总裁办公室里的程序员”。

後來，在公司的系统里，我的职位写得很正式：

总裁办高级软件工程师。

我喜歡這個有特色的名字。

\
丁磊很少真正坐在办公室里。

他经常在公司各个地方转。

有时候在游戏部门，有时候在会议室。

很多时候，已经很晚了，他还在公司。

有一次我晚上还在楼里，看见他从会议室出来。

走廊很安静，只剩下一些电脑屏幕的光。

还有一天下午，我们一起出去吃饭。

不是去什么特别的地方，只是工业园附近的一家小苍蝇馆子。

广州的空气有点热。

我们坐在门口的塑料凳上。

靠近门口有一口大锅，老板在下面条。

热气一阵一阵往外冒，混着酱油和葱花的味道。

丁磊穿得很普通，和在公司时差不多。

我们各要了一碗面。

他问我习不习惯广州，我说还行。

他又说，公司现在人多了，很多事情他管不过来，你好好做就行。

话不多。

面吃得很慢。

旁边有几桌人，大声说着粤语，我们这边反而很安静。

吃完以后，我抢着把单买了。

他是首富。

但我觉得这顿饭应该我请。

他也没有和我争，只是说了声谢谢。

然后我们一起往回走。

两个人并排走，但谁也没有再提公司的事。

那是我第一次和他在公司外面吃饭。

也是印象最深的一次。

后来他让我做游戏。

那几年网易的游戏刚刚开始起来，很多人挤破头去做。

但我当时觉得，自己是研究生毕业，去做游戏好像有点“不务正业”。

现在回头看，这个想法其实很幼稚。

那时候我还想继续读书。

有一次我跟他说，我想出国读博士。

他直接回了句：

书读多了没用。

我当时也认同。

毕竟他只是本科毕业，却做出了这么大的事业。

于是我也没有再继续申请。

关于介绍女朋友，我几次想问他，但發現當面不太好意思開口。

尤其公司人那麼多的時候。

过了几天，公司有人来找我，说安排我见一个人。

见面以后我才发现，是一位似乎比我还小点的女上司。

我当时以为，这大概就是他给我介绍的人。

过了段时间才知道，人家早就已经结婚了。

那时候我有点生气。

觉得他说话不算数。

现在回头看，那时候我其实很不成熟。

我把公司当成了学校了。

\
我在网易待的时间不算很长。

那几年我一直在想一个问题：

人为什么要工作。

不是为了挣钱，而是更深一点的东西。

我总觉得，世界应该更大一点。

所以后来我离开了网易。

离开的时候，没有什么特别的场面。

我对丁磊说，三年以后再回来。

他抽了口雪茄，点头说好。

我收拾好东西，从办公室走出来。

走廊还是一样安静。

很多人还在写代码。

很多事情还在继续。

但有一件东西留在了我心里。

那就是那幅字。

云层之上无风雨。

很多年以后，我还会想起那间办公室。

想起我坐在那里写代码，偶尔抬头，看见墙上的那五个字。

互联网把我带进了那间办公室。

后来又把我带去了很多地方。

但那幅字，一直没有离开过。

[^1]: 丁磊，网易创始人，2003 年前后曾为中国首富。

[^2]: Flash，Adobe 推出的多媒体创作与播放平台，2000 年代曾广泛用于网页动画、游戏与互动内容，2020 年底已停止更新与支持。

[^3]: MSN Messenger（后更名为 Windows Live Messenger），微软推出的即时通讯软件，2000 年代在中国及全球广泛使用，2013 年除中国大陆外多数地区已关闭，2014 年中国大陆服务亦停止。
