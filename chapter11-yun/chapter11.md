# 第十一章 云

> 云在上面，人在下面。

后来我离开了 Uber。那几年 Uber 的自动驾驶项目很热，很多工程师相信车很快就可以自己开，人只需要坐在里面。后来出了事故，再后来，公司把自动驾驶业务卖给了 Aurora。很多人觉得这是一个商业决定，但事故改变了很多东西。我当时心里有一点不舒服——有些事情你以为会一直做下去，忽然就结束了。

那段时间我开始重新看机会。面前摆着几条路：Apple Music 的 offer 已经到手，前组里一半同事都去了那里；一家大银行也抛来了橄榄枝；还有 Oracle。Oracle 是最不起眼的那个。那时候它的股价在五十块左右徘徊了很多年，很多人觉得这是一家老公司，技术世界的注意力已经在别的地方。Apple 的 hiring manager 甚至直接问我：那公司都快死了，你去干什么？但我还是去了。

我去面试的时候，有一件事情让我印象很深。面试我的人，聊起来才发现是大学的师兄。我们本来不认识，但说着说着，发现都认识一些人，同一个系，还跟过同一位老师，就对上了。我们聊了很久，从共同的导师聊到校园里的树影，忽然想起很多大学里的事、很多人、还有那时候的自己。后来 hiring manager 也和我聊了很久，很热情，不像面试，更像是在认真地介绍一个团队。那天面试结束以后，我忽然觉得，也许可以试一试。那天晚上我睡不着。

入职的第一周，落后的系统和沉闷的氛围让我几乎想走。我拨通了之前那家无人车公司 HR 的电话，想问问能不能回去。对方在电话那头说：你既然选择了 Oracle，说明你已经想去养老了，你的价值观和我们不符。她不仅拒绝了我，还收回了之前的邀请。那是我职业生涯里很郁闷的一刻。被当成「想养老的人」，对一个还在做分布式系统和AI的我来说，像一记闷棍。我对自己说，既然没退路了，那就把该做的事做完。撞了南墙，也要看看墙后面是什么。

后来我开始在 Oracle 做云上的 AI。OCI，Oracle Cloud Infrastructure。数据中心一排一排机器，服务器在机架里亮着灯，风扇一直在转，声音像远处的海。很多工程师每天写代码、写系统、写模型。云其实不是在天上，是在世界各地的数据中心里，机器一排一排摆在那里又重又熱，但我們叫它「云」。

在 ChatGPT 火起来之前，我就在做大語言模型。那时候这个领域已经热了，但还没有席卷所有人。我在 GitHub 上随手发了一个 repo，很快就有上百人点赞。私信里什么人都有：医生、律师、大学生。有人说要开车到我家来学。那种狂热让我有一点怕——像大家在黑暗里预感火要来了，拼命想抓住那点光。

到了 2023 年 1 月，ChatGPT 火了。我站在机房外，听着风扇像海浪一样响，心里只有一个念头：技术拐点到了。山雨欲来风满楼。我们以前做的很多传统模型，可能都要推倒重来。

后来我做Copilot。名义上是帮程序员干活，把人从繁琐的代码里解放出来。但我心里清楚，我们的目标，最终是让 AI 替代程序员。那是一种说不出的滋味。我每天看着身边的同事，敲键盘、讨论架构。在我眼里，他们好像渐渐变成一行行冰冷的代碼、一块块发烫的 GPU。而我，就是参与做这件事的人。我无路可退，他们也馬上會无路可退。有一次，我测出一套系统能替代大约 70% 的初级开发工作。那天大家在庆祝，在欢呼性能的飞跃。我坐在边上，心想：这是不是意味着很快就会有 70% 的同行被机器替代？他们被替代之后，我们呢？以后的代码还要人写吗？活还要人干吗？我们做的是挖坑埋自己的活，而且为了不被时代抛下，还不能埋得太慢。

有一次我见到了 Larry Ellison。他已经七十多岁了，走路很慢，但说话很直接。很多工程师围在旁边听他说事情。技术世界也在变：AI 模型越来越大，代码越来越多，机器开始做以前只有人才能做的事情——写程序、写文档、回答问题。有时候我会想一个很简单的问题：如果机器也能写代码，人为什么还要写？如果机器也能思考，人为什么还要工作？云在上面，人在下面，代码在中间。你写代码，机器也在写代码。

很多时候我会想起很久以前看到的一句话。那是在广州的一间办公室里，墙上挂着五个字：云层之上无风雨。很多年以后，我真的开始在「云」里工作。但云层之下，人还是要走路、要奔波、要做决定。机器在学习，系统在运行，服务器在很远的地方，风扇一直在转，数据在世界各地流动。有时候机器写得比人还快。云层之上或许无风雨。云层之下，我们亲手参与了一场会改变很多人的风暴。

云在头顶。

人还在走路。

路，却不知道通往哪里。
