# 《云层之上无风雨》总结与写作指南

## 一、全书总结

**书名**：云层之上无风雨（来自网易时期丁磊办公室那幅字）

**体裁**：自传体小说，约 95% 真实，类似《阿甘正传》——个人成长 + 时代见证，主题积极。

**核心主题**：
- 一人一时代：从山野到云端，在洪流中活成自己的见证。
- 命运不是被规划的路径，而是在时代里不断打开的疆域。

**人生弧线**：农村 → 小镇（简阳中学）→ 大学（美术社、合唱团、99 国庆）→ 留学（新加坡、MIT/SMA）→ 回国进网易（丁磊、「云层之上无风雨」）→ 离开 → 华尔街 / 新交所交易员 → Uber 无人车 → OCI 云上 AI → 无码时代与生命反思（无风雨、无常、虚无）。

**结构**：序章 + 12 章，每章一字标题，形成一条线索：**我 → 山 → 路 → 门 → 火 → 歌 → 旗 → 海 → 网 → 街 → 谷 → 云 → 无**。

## 三、怎么写

### 1. 素材从哪里来

- **主素材**：`raw/1.txt`，中英混杂的回忆、细节、金句，按章节主题去「认领」到对应章。
- 每章动笔前：在 `raw/1.txt` 里搜关键词（丁磊、网易、Uber、简阳、落榜、合唱团、彭丽媛、托福、SMA、新加坡……），把相关段落复制到章提纲里，再按时间或主题排顺序。

### 2. 语气与视角

- **第一人称**，像阿甘：简单、具体、少议论，用经历和细节带出命运感。
- 不煽情、不卖惨：即使写落榜、穷、失恋，也保持「当时就是这样，后来才懂」的平静。
- 时代感靠**具体事**：国庆游行、网易 2003、首富、火锅店老板后来成首富、Gilbert Strang、丁磊办公室那幅字……不必解释背景，让读者自己感受。

### 3. 每章可以怎么写

- **序章 我**：极短。断奶 = 第一次有「我」；两岁见雪 = 第一次被世界震撼。几百字即可，像开场白。
- **山 / 路 / 门**：童年与少年。多写感官（泉水、土路、校门、石灰岩被偷、泥墙雕塑）。路：那条土路 + 「为什么赢不了」的念头反复出现。门：修门的一千块、鲤鱼跃龙门、寒门进简阳中学。
- **火 / 歌 / 旗**：高中与大学。火：状元、火锅店、后来的首富。歌：美术社、书法、合唱团、校花团长、美声。旗：99 国庆、天安门、彭丽媛、江主席——时代大事件用个人视角（谁带队、你站在哪、看到什么）。
- **海**：出国一条线。新加坡、托福 GRE（帮人考、新东方贴广告）、MIT/SMA、Gilbert Strang、PhD 未读。可写「海」的意象：离乡、渡海、另一种人生。
- **网**：回国、写文章、被丁磊看到、进网易、办公室那幅字「云层之上无风雨」、首富还加班到凌晨、小馆子吃面你抢着买单。离开的原因：不成熟、想多看世界、以及那件答应介绍对象却安排了已婚女上司的事——用细节带过，不审判。
- **街**：离开网易 → 华尔街对冲基金、新交所交易员。写「街」：金融、交易、执照、另一种节奏。
- **谷**：Uber，当时宇宙第一 startup，无人车。写硅谷、写造车、写那个时代的狂热与可能。
- **云**：OCI，在云上做 AI。写云、写基础设施、写从代码到 AI 的转变。
- **无**：收束。无码（从写代码到 AI 会写代码）、无风雨（回应书名）、无常（母亲生病）、虚无（青春不在、时间流逝）。最后落在反思生命的意义，不必给答案，留白即可。

### 4. 单章篇幅建议

- 序章：500–800 字。
- 第 1–6 章（成长）：每章 3000–5000 字，可长可短看素材。
- 第 7–12 章（职场与收束）：每章 2500–4000 字，节奏可稍紧，最后「无」可多留空间给感受与思考。

### 5. 技术细节（与 readme 一致）

- 每章一个文件：`chapterN-xxx/chapterN.md`，序章用 `chapterx/preface.md`。
- 当前目录名还是旧标题（如 chapter3-luo、chapter9-xue），若要与新标题一致，可逐步改为 chapter3-men、chapter9-jie、chapter12-wu 等；不改也能写，build 时按顺序认 chapter1…12 即可。
- 写完一批章后用 bubble 生成 PDF 检查效果：`bubble-build --lang cn --max-chapters 12`。

## 四、建议的写作顺序

1. **先补序章**：把「雪」改成「我」，断奶 + 两岁见雪，定调。
2. **挑最有画面的一章开写**：比如 山、门、网、无，写顺了再往前/后推。
3. **每章先列「必写清单」**：从 readme 表 + raw/1.txt 摘出 5–10 个必写点，再扩成文。
4. **最后统稿**：检查「云层之上无风雨」在书中出现的位置（网易办公室、结尾无风雨），以及「我→山→…→无」这条线是否清晰。

---

你已有 raw 素材和清晰结构，接下来就是按章「认领」素材、用第一人称把一件事一件事写实、写短句、少解释，让时代和命运从细节里自己浮出来。
